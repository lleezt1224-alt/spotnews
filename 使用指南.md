# 香港零售「回暖」與結業潮並存 - 數據新聞網站

## 🎯 網站特色

### 設計感元素
- **綠色主題**：代表零售業的活力和回暖趨勢，配合深綠漸變背景
- **紅色強調**：用於負面數據和下跌指標，形成視覺對比
- **現代漸變**：導覽欄和數據看板使用線性漸變效果
- **玻璃態樣式**：數據卡片使用毛玻璃效果（backdrop-filter）
- **品牌色彩應用**：統一的色彩系統貫穿全頁

### 互動性功能
1. **導覽欄活躍指示**：當用戶滾動時，導覽欄自動標記當前章節
2. **數據卡片互動**：
   - 懸停時發出通知
   - 點擊顯示數據訊息
   - 悸動的視覺反饋
3. **平滑滾動**：所有導覽連結支持平滑過渡
4. **圖表放大查看**：點按圖表可彈出放大版本
5. **回到頂部按鈕**：固定位置按鈕方便用戶快速返回
6. **懶加載動畫**：內容進入視窗時觸發淡入動畫
7. **統計數字計數**：英雄區域的數字會有動畫效果

### 響應式設計
- 完全適配桌面、平板和手機
- 媒體查詢斷點：768px 和 480px
- 觸控友好的按鈕和連結

## 💡 使用建議

### 部署方法
1. 將所有文件放在同一文件夾中
2. 確保圖片文件與HTML在同一目錄
3. 用瀏覽器打開 `index.html`

## 🎨 自定義色彩

如要更改主題色，編輯 `style.css` 中的 `:root` 部分：

```css
:root {
    --primary-color: #2d5a3d;      /* 深綠 */
    --secondary-color: #d63031;    /* 紅色 */
    --accent-color: #00b894;       /* 亮綠 */
    --text-dark: #2c3e50;          /* 深灰 */
    --text-light: #7f8c8d;         /* 淺灰 */
    --bg-light: #f8f9fa;           /* 極淺灰 */
}
```

## 📞 故障排除

**問題**：圖片無法顯示
- 確保圖片與 HTML 在同一文件夾
- 檢查圖片文件名是否正確
- 確保文件格式是 .png 或 .jpg

**問題**：樣式未加載
- 清除瀏覽器快取（Ctrl+Shift+Delete）
- 檢查 CSS 文件是否與 HTML 在同一文件夾
- 確保 `<link rel="stylesheet" href="style.css">` 正確

**問題**：JavaScript 功能不工作
- 打開瀏覽器控制台查看錯誤信息（F12）
- 確保 script.js 文件在正確位置
- 檢查是否有 JavaScript 錯誤
